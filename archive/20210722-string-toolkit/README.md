# string_toolkit.h
A simple C/C++ library containing tools to manipulate a C string. Inspired by [string-toolkit by VoltrexMaster](https://github.com/VoltrexMaster/string-toolkit). Everything is contained in the header file so you can just download it.

- [C header documentation](https://github.com/null8626/string_toolkit.h/blob/main/README_C.md)
- [C++ header documentation](https://github.com/null8626/string_toolkit.h/blob/main/README_CPP.md)
- [Python wrapper documentation](https://github.com/null8626/string_toolkit.h/blob/main/README_PYTHON.md)
